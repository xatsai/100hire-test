Google just rolled out a feature in Search Console that every SEO should set up today.

You can now filter branded vs. non-branded queries natively. No more regex hacks or manual keyword lists.

It just expanded to all eligible sites this week.

Why this matters:

- Branded traffic = people who already know you.
- Non-branded traffic = people discovering you for the first time.

If you're not separating these, you're flying blind on what's actually driving growth.

Here's how to use it right now:

1. Open Google Search Console → Performance report
2. Click "Search type" → look for the new branded/non-branded filter
3. Compare the two segments side by side

What to look for:

- If branded traffic is growing but non-branded is flat → your content strategy isn't working
- If non-branded is growing but branded is flat → you're attracting visitors but not building a brand
- If both are growing → you're winning

⚠️ One heads up: check the lists Google creates. 

Some agencies found their own brand name incorrectly classified as "non-branded." 

Verify accuracy before making decisions off the data.

Data only goes back to February 20th, so give it a few weeks to build up.

But set it up now. The sooner you start segmenting, the faster you'll spot what's working.
